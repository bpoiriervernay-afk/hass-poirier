# Web Scraper – Home Assistant

Intégration Home Assistant permettant de créer des capteurs de scraping web
via CSS ou XPath, configurables depuis l’UI.

## Fonctionnalités
- Configuration UI
- CSS / XPath
- Fréquence configurable
- Bouton Force Update
- Statut visible dans l’entité

## Installation via HACS
1. HACS → Intégrations
2. Dépôts personnalisés
3. Ajouter ce repo
